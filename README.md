# 🧮 Student Grade Calculator (CLI)

A simple Node.js command-line application that calculates a student's total marks, percentage, and grade based on marks in 3 subjects.

## 🚀 How to Run

1. Open terminal in your project folder
2. Run this command:
```bash
node calculator.js

🎓 Grading System
Percentage	Grade
90% and above	A
75% - 89%	B
50% - 74%	C
Below 50%	Fail
